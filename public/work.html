Shark Store - E-commerce Website

Shark store is an e-commerce application. The store includes men clothes, women clothes and sneakers. I only displayed the "trending" items in the homepage, and all the other items are in the shop page. You can add items to cart and go to checkout page and even use a fake payment card to see that stripe works.

I created this application using React JS and firebase. I used Redux and Redux toolkit to manage state in the application, and I stored items in the localStorage.

myTasks

This is a tasks management application that you can use to create tasks, update them and also delete them, basically a CRUD application. To create new tasks you only need to login to your account or create a new one and go to your tasks page and start creating tasks. The tasks that you create are stored in the database and you can access them whenever you want.

I created this application with React JS in the front-end and express and mongoose in the back-end. To secure the users information I used cookies to store tokens, and jsonwebtoken to generate them. I also used bcrypt to hash the password.
